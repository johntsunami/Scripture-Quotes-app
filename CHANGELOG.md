# Changelog

All notable changes to this project are documented here.

The format is loosely based on [Keep a Changelog](https://keepachangelog.com/),
and the project follows semantic versioning where practical.

## [0.7.2] — 2026-05-13

### Added
- **`scripts/update-from-zip.ps1`** — smart update workflow that replaces
  the old wipe-and-extract flow. Stops the running app, extracts to a temp
  folder, mirrors source files into the project while preserving `.git/`,
  `data/`, and `node_modules/`. Then runs setup, shows the Git diff, and
  offers to push and start the app.
- **`.gitattributes`** — silences "LF will be replaced by CRLF" warnings
  by telling Git to normalize line endings consistently across platforms.

### Removed
- Dead `src/main/quote-store.js` file (replaced by `deck-store.js` in v0.6.0
  but was still present in the install layout).

## [0.7.1] — 2026-05-13

### Added
- GitHub-ready repository structure: comprehensive `.gitignore`, this changelog,
  improved README with clone-and-run instructions.
- `scripts/push-to-github.ps1` helper that bumps version notes, commits, and
  pushes to GitHub in one command after extracting a new zip.

## [0.7.0] — 2026-05-13

### Added
- **Auto-start with Windows.** New default-on toggle in Settings → Startup.
  Registers the app under `HKCU\Software\Microsoft\Windows\CurrentVersion\Run`
  via Electron's `app.setLoginItemSettings`. Passes `--hidden` on auto-launch
  so the boot popup is suppressed; the normal scheduled timer takes over.
- **Show a quote on launch.** New default-on toggle: every time you open the
  app, a popup appears within ~1.5 seconds (skipped during quiet hours and on
  silent auto-boots).
- **Startup section in Settings** holding both new toggles.

### Changed
- **Default interval is now 30 minutes** (was 60).
- **Quiet hours are now ENABLED by default**, 22:00 → 07:00. Users can
  disable in Settings.

## [0.6.1] — 2026-05-13

### Fixed
- **Popup action buttons (Dismiss / Save / Delete) were invisible** because
  `.quote__actions` had `position: absolute` with no positioned ancestor and
  no horizontal anchor, rendering them off-screen. Added `position: relative`
  to `.quote` and explicit `left: 50%; transform: translateX(-50%)` on the
  actions block. Also added `pointer-events: none` until hover so the hidden
  buttons don't block clicks on the quote text.

## [0.6.0] — 2026-05-13

### Added
- **Multi-deck storage.** Quotes are now organized into separate decks. Each
  deck lives in its own JSON file under `data/decks/`. Built-in decks ship in
  `assets/decks/` and are copied to `data/decks/` on first launch.
- **Round-robin rotation.** When multiple decks are enabled, popups cycle
  through them one at a time. Each deck tracks its own seen-queue so toggling
  decks doesn't reset rotation progress.
- **Bible — KJV Essentials deck**: 436 hand-curated verses across all 66
  books, public domain. Generated by `scripts/build_bible_kjv_deck.py` for
  reproducibility and easy extension.
- **Decks section in Settings** with toggles, rename, delete, and "+ New
  empty deck" button. Built-in decks show a badge and can be disabled but
  not renamed or deleted.
- **Library multi-select**: click rows to select, shift-click for range,
  bulk Favorite / Unfavorite / Delete actions.
- **+ Add a single quote** inline form in the Library section.
- **Import target dropdown** in step 3 of the AI prompt flow — pick existing
  user deck or "+ New deck (named from topic)".
- **Caps with friendly error messages**: 20 decks max, 5000 quotes/deck,
  50000 total, 500 chars/quote.
- **Legacy migration**: old `quotes.json` from previous versions is moved
  into a deck named "Imported" on first launch.

### Removed
- The old "Categories in rotation" feature (replaced by per-deck enable
  toggles, which match how people actually think about their content).
